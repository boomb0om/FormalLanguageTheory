## Лабораторная №3

Построение нормальной формы Грейбах методом устранения левой рекурсии и методом Блюма–Коха

## Синтаксис входных данных (в [РБНФ](https://ru.wikipedia.org/wiki/%D0%A0%D0%B0%D1%81%D1%88%D0%B8%D1%80%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%BC%D0%B0_%D0%91%D1%8D%D0%BA%D1%83%D1%81%D0%B0_%E2%80%94_%D0%9D%D0%B0%D1%83%D1%80%D0%B0)):

```
grammar = rule {rule}
rule = nterm -> { (term | nterm) }
term = (A–z) | (0–9)
nterm = (A–z {A–z} {0–9})
```

Пример правильных входных данных:
```
[S]->[X][A]
[S]->[B][B]
[B]->b
[B]->[S][B]
[X]->b
[A]->a
```

## Установка
Необходимо установить [graphviz](https://graphviz.org/download/). Он используется для отрисовки автоматов в графы.
```bash
git clone https://github.com/boomb0om/FormalLanguageTheory
cd FormalLanguageTheory/lab3
pip install -r requirements.txt
```
## Запуск
Запустить лабу на тесте (по-умолчанию используется алгоритм устранения левой рекурсии):

1) Создайте текстовый файл с данными в нужном синтаксисе
2) Запустите `main.py` и перенаправьте в него поток чтения из файла с данными
    ```python
    python main.py < tests/gnf.txt
    ```
Возможные ключи:
1. `--debug` - вывод промежуточных действий
2. `--blum-coh` - использовать алгоритм Блюма-Коха
3. `--graphs` - сохранять графы автоматов (работает только с алгоритмом Блюма-Коха)

Программа выведет новую грамматику в ГНФ, результат проверки на ГНФ, время выполнения, а также:
1. Для метода устранения левой рекурсии выведет частичные порядки нетерминалов
2. Для метода Блюма-Коха автоматы сентенциальных форм для каждого нетерминала
3. В случае если установлен ключ `--graphs`, то автоматы сентенциальных форм будут сохранены в виде изображений в папку `graphs/`
4. В режими дебага также выводятся грамматики после промежуточных действий, а также пояснения к этим самым действиям
